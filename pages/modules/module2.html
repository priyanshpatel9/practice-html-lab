Root Relative File Paths

A root relative file path is a path that starts from the root of the website. It always starts with a forward slash /.

For example, /images/logo.png is a root relative file path.

Notice that's what I used above in the anchor tags in the navigation menu. This is the best way to reference files and resources on your website.
